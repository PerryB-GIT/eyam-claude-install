# Claude Code - Eyam Health Install

## One command in Terminal:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/PerryB-GIT/eyam-claude-install/main/install.sh)
```

## What it does
- Installs Claude Code via npm
- Takes about 30 seconds

## After install
1. Open a **new Terminal window**
2. Type `claude` and press Enter
3. Enter your Anthropic API key when prompted
