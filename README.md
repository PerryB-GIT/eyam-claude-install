## Step 1 — Install Claude Code

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/PerryB-GIT/eyam-claude-install/master/install.sh)
```

After install: open a new Terminal, type `claude`, enter your Anthropic API key.

## Step 2 — Your Agents

These are your 4 custom AI agents, built for your specific role and workflows.

| Agent | File | What It Does |
|-------|------|--------------|
| Meeting Agenda & Debrief | `agents/meeting-agent.md` | Turns raw notes or dictation into structured agendas and debriefs |
| Grant & Partnership Tracker | `agents/grant-tracker-agent.md` | Tracks funding opportunities, milestones, and deadlines |
| Follow-Up Summarizer | `agents/followup-agent.md` | Extracts and organizes follow-ups from meetings and emails |
| Monday.com CRM | `agents/monday-crm-agent.md` | Builds and maintains your stakeholder CRM |

---

## How to Use an Agent

1. Open Terminal and type `claude`
2. Open the agent file you want (click any file above on GitHub)
3. Copy the entire contents
4. Paste it into Claude and press Enter
5. Claude is now in that agent mode — give it your notes, emails, or dictation
